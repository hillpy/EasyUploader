## 安装

### NPM方式

* 执行以下命令即可

    ```
        npm install easyuploader --save
    ```

### script引入方式

* 使用unpkg的cdn

    ```
        <script src="https://unpkg.com/easyuploader/dist/easyuploader.min.js"></script>
    ```
    <br/>
    ```
        https://unpkg.com/easyuploader/dist/easyuploader.min.js
    ```

* 直接将dist目录下的easyuploader.min.js拷贝至您的项目目录中并引入即可
