# 绑定类选项

插件需绑定到元素按钮节点或者input file上才可以使用，选项如下：

## el

* 类型：String
* 默认值：""
* 值示例："el": "#btn"
* 解释：该选项表示上传功能需要绑定的元素按钮节点。可以是button元素，或者div元素等等。当该选项存在时file选项将无效。值建议为"#id"形式而非".class"形式。

# file

* 类型：String
* 默认值："#file"
* 值示例："file": "#file"
* 解释：该选项表示上传功能需要绑定的元素按钮节点。可以是button元素，或者div元素等等。当该选项存在时file选项将无效。值建议为"#id"形式而非".class"形式。
